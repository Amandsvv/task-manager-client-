#  Frontend – React + Vite + Tailwind

## Overview

Minimal frontend interface built to interact with the backend REST APIs.

It demonstrates authentication flow and protected CRUD operations.

## Tech Stack

* React (Vite)
* TailwindCSS
* Axios
* React Router

## Features

* User Registration & Login
* JWT cookie-based authentication
* Protected dashboard
* Task CRUD operations

## Setup

```bash
npm install
npm run dev
```

Runs at:
http://localhost:5173
